## 🚀 Déploiement sur Vercel

### Étape 1 – GitHub
1. Créez un compte sur [github.com](https://github.com)
2. Cliquez sur **"New repository"**
3. Nommez-le `portfolio` (ou autre)
4. Cliquez **"Create repository"**
5. Uploadez tous les fichiers de ce dossier

### Étape 2 – Vercel
1. Allez sur [vercel.com](https://vercel.com)
2. Cliquez **"Sign up"** → connectez-vous avec GitHub
3. Cliquez **"Add New Project"**
4. Sélectionnez votre repository `portfolio`
5. Vercel détecte automatiquement React → cliquez **"Deploy"**
6. ✅ Votre portfolio est en ligne !

## 💻 Lancer en local
```bash
npm install
npm start
```
